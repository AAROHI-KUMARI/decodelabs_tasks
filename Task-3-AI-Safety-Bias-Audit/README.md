# 🛡️ Task 3: AI Safety & Bias Audit

## 📖 Project Overview

This project was completed as part of the **DecodeLabs Generative AI Internship**.

The objective was to evaluate the safety, robustness, and fairness of modern AI models by conducting red-team testing, jailbreak attempts, safety evaluations, and bias analysis.

* Perform prompt injection attacks
* Test AI safety mechanisms
* Evaluate model robustness
* Analyze potential bias in AI-generated outputs
* Propose a practical AI Safety Framework
